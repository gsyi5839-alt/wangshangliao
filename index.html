系统要求

请确保您的设备满足以下最低要求

Windows

  • Windows 10 (64位) 或更高版本
  • .NET Framework 4.7.2 或更高版本
  • 4GB 内存（推荐 8GB）
  • 200MB 可用磁盘空间
  • 稳定的网络连接

macOS

  • macOS 12 (Monterey) 或更高版本
  • 支持 Intel 和 Apple Silicon (M1/M2)
  • 4GB 内存（推荐 8GB）
  • 200MB 可用磁盘空间
  • 稳定的网络连接
